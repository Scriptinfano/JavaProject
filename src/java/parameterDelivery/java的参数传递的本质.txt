  1、java中的参数传递本质上都是按值传递，只不过值的内容是对象的引用
  2、java方法传递基本数据类型时，由于基本数据类型分配在栈上，因此是按值传递，对参数所作的任何修改不影响原值
  3、java方法传递类对象时，本质上也是按值传递，不过值的内容是对象的引用



  打比方：java传递类对象时，好比将房子的钥匙复制了一份传给参数，参数拿着复制的钥匙开原有的房子
    而C++的按值传递都是把整个房子复制一份交给参数，C++的按引用传递也和java不一样，C++的引用传递
    相当于是把自己房子的钥匙直接交给参数而不是把钥匙复制一份，也就是说java里的引用像是房子的钥匙
    而C++的引用则更像是房子本身java中的传引用更类似于C++的传指针，和C++中的传引用有根本的不同
